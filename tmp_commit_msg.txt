Mobile bundle Phase 2.5: About/Manual/Changelog i18n across all 23 languages

### Added
- TranslationCatalog: 6 new top-level string fields (aboutTitle, manualTitle, changelogTitle, gplLicenseTextFallback, manualTextFallback, changelogTextFallback) covering all 23 supported languages.
